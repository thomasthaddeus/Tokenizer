Help and Support

Getting Started

Welcome to the Text Truncator with File Upload application. This tool allows you to easily manipulate and truncate your text files according to your preferences.

How to Upload Files

To upload a text file for processing:

  1. Click the "Choose File" button on the sidebar.
  2. Select the text file from your computer.
  3. The content of the file will be displayed in the input area.

How to Use Cleaning Options

The cleaning options allow you to modify the text by removing certain elements or changing its case:

  • Convert to Lowercase: Transforms all characters to lowercase.
  • Remove Special Characters: Eliminates characters like @, #, $, etc.
  • Remove Numbers: Removes all numeric characters.
  • Remove Punctuation: Strips punctuation marks like ., !, ?, etc.
  • Remove Emojis: Clears out all emoji characters.

How to Truncate Text

To truncate your text:

  1. Enter the maximum number of tokens in the "Max tokens" field.
  2. Click "Truncate Text". The truncated text will appear in the corresponding area.

Frequently Asked Questions

What is a token?

A token typically refers to words in the context of this application.

Is there a limit to the size of the file I can upload?

There is no set limit, but very large files may take longer to process and can impact performance.
